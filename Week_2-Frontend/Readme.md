### Day 1 ‚Äì Semantic HTML

---

### Day 2 ‚Äì CSS Layout (Flexbox & Grid)

* Built complex layouts using:

  * **Flexbox** for one-dimensional layouts (header, sidebar)
  * **CSS Grid** for two-dimensional layouts (content sections)
* Created a centered wrapper layout
* Implemented **mobile-first responsive design**

**Learning:**
Flexbox is best for alignment in one direction, while Grid is ideal for structured layouts with rows and columns.

---

### Day 3 ‚Äì JavaScript & DOM Manipulation

* Practiced modern JavaScript (ES6+)
* Built an interactive **FAQ Accordion**
* Learned:

  * Event listeners
  * DOM traversal
  * Managing UI state without frameworks

**Learning:**
UI behavior should be driven by application state, not direct DOM manipulation hacks.

---

### Day 4 ‚Äì JavaScript Utilities & LocalStorage

* Built a **Todo App** with:
